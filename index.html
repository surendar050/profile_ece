Arduino Embedded C YOLO OpenCV Python UART

Density-Based Adaptive Traffic Light Control System

A hardware-software co-design system that replaces fixed-timer traffic signals with YOLO-powered computer vision feeding an Arduino microcontroller — physical lights respond dynamically to real vehicle density.

  • YOLOv5 object detection counts vehicles per lane on live camera feeds in real time
  • Arduino (Embedded C) receives lane density data and modulates traffic light timings via digital I/O & PWM
  • Python serial bridge (UART) translates YOLO inference output into hardware control commands
  • OpenCV handles frame capture, pre-processing, and ROI-based lane segmentation
  • Demonstrates: microcontroller I/O, embedded programming, real-time CV, sensor-to-actuator control loops
